# Rupa

Rupa는 실내 볼더링에서 막힌 동작을 벽 사진 위에서 직접 시험해보는 볼더링 시뮬레이터입니다.

벽 앞에서 바로 시도하기 전에, 사진 위에서 내 몸의 위치를 옮겨보며 가능한 자세를 먼저 탐색할 수 있습니다.

## 왜 만들었나요?

볼더링을 하다 보면 “다음 홀드를 어떻게 잡아야 하지?”보다 “내 몸을 어디에 둬야 하지?”에서 막히는 순간이 많습니다.

Rupa는 정답 무브를 추천하지 않습니다. 대신 사용자가 직접 스타트 홀드, 탑 홀드, 몸의 자세를 맞춰보면서 자신의 움직임을 실험할 수 있게 돕습니다.

## 무엇이 아닌가요?

- 등반 기록 앱이 아닙니다.
- 커뮤니티 앱이 아닙니다.
- AI가 정답 무브를 추천하는 앱이 아닙니다.

Rupa의 중심은 “사진 위에서 직접 움직임을 시험해보는 경험”입니다.

## 핵심 흐름

```text
벽 사진 선택
-> 홀드 분석
-> 스타트 홀드 선택
-> 홀드 조정
-> 탑 홀드 선택
-> 몸 크기 조정
-> 자세 시뮬레이션
```

사용자는 인식되지 않은 홀드를 직접 추가하고, 탑 홀드를 고른 뒤, 스켈레톤을 움직여 완등 자세를 만들어볼 수 있습니다.

## 주요 기능

- 벽 사진 촬영 또는 갤러리 선택
- 홀드와 같은 색 루트 분석
- 스타트 홀드와 탑 홀드 선택
- 인식 누락 홀드 수동 조정
- 사용자 신체 정보 기반 스켈레톤 조작
- 양손이 탑 홀드 근처에 도달했을 때 완료 상태 표시

## 기술 개요

Rupa는 하나의 제품 안에 모바일 앱, API, 비전 서비스를 함께 둡니다.

- 모바일 앱: Expo React Native, Expo Router, React Native SVG, Zustand, TypeScript
- API: NestJS
- 비전 서비스: FastAPI, OpenCV, YOLO, Roboflow fallback

모바일 앱은 사용자 경험과 로컬 시뮬레이션 상태를 담당하고, Nest API는 앱과 비전 서비스 사이의 안정적인 경계를 맡습니다. FastAPI 비전 서비스는 벽 사진 분석과 루트 탐지를 담당합니다.
